# Ready Runner One

A small fullscreen browser runner inspired by classic pixel arcade games.

Reach `10000` points and find the key.

## Play locally

Open `index.html` in any modern browser.

## Controls

- `Space` - start / jump / restart

## GitHub Pages

1. Create a repository named `ready-runner-one`.
2. Upload all files from this folder to the repository root.
3. Open `Settings` -> `Pages`.
4. In `Build and deployment`, choose:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
5. Save and wait until GitHub publishes the site.

Your site will be available at:

```text
https://YOUR_USERNAME.github.io/ready-runner-one/
```
